# 🔐 Phishing Awareness Training Website

A simple educational web project designed to spread awareness about phishing attacks, fake websites, and social engineering techniques.  
This project was built as part of the CodeAlpha Cyber Security Internship.

---

## 🚀 Live Demo
🔗 https://sarveshwar-v.github.io/CodeAlpha_PhishingAwarenessTraining/

---

## 📌 Features
- 📧 Learn about phishing emails and how to identify them  
- 🌐 Understand fake websites and scam detection techniques  
- 🧠 Awareness of social engineering attacks  
- 🎯 Interactive quiz for self-assessment  
- 📱 Multi-page responsive website structure  

---

## 🛠 Tech Stack
- HTML5  
- CSS3  
- JavaScript (basic interactivity)
